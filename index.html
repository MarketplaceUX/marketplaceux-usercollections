Firefox Marketplace User Collections Specs

This site contains the design specifications for the Firefox Marketplace's User Collections. This document is currently a living document as the design process is underway.

Get started by reading the Overview »

If you would like to contribute to the design, please file an issue against this document using the link in the top navigation or feel free to clone and make pull requests against the document at: the spec GitHub repo. All the individual design artefacts that are generated to create this spec can be found in the master branch of this repo.
