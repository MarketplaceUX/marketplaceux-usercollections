My Account Page → Delete Collection

Mobile

  1. Collection edit page
    • The Delete collection button should always be at the bottom of the collection.
    • When the Delete collection button is selected, the delete confirmation appears.
  2. Delete confirmation
    • The user must confirm that the collection should be deleted.
    • Upon confirmation, return the user to the "My Collections" screen.
  3. My collections screen, with confirmation message
    The confirmation message should be “Collection deleted”.
