Home Page → Community Page

The Community page is where users can view collections created by community members who have chosen to make their collections public.

Community Page: Mobile

  1. Main Navigation Bar
    "Community" is a main navigation item listed after "Categories"
  2. Popular and New Tabs
    User can toggle between viewing user collections listed in order of most popular and new.
  3. User Collections – What To Show?
    • By default, the user should view the most popular order of collections
    • Only show user collections that have 3 or more apps
    • If none of the apps in the collection are available in the user's region, the user's operator, or on their device (mobile use case) the user should not see that collection.
    • If an app in a collection is not available in the user's region, by the user's operator, or on their device (mobile use case), the user will not see that app though they will be able to see the rest of the apps in the collection
  4. Sharing and Upvoting
    • The user should be able to share a collection.
    • The user should be able to upvote a collection.

Community Page: Desktop

Acceptance criteria for the community page on desktop

  1. Users Collections are visible under the “Community” tab in the main navigation
  2. Only user collections that follow the visibility rules will be shown.
  3. On desktop the sections of the page have titles
  4. By default collections should be sorted by popular. The user can also sort collections by “new.”
  5. The create new collection call-to-action will take user to the create a collection screen.
  6. Collections are listed from most recently created to oldest in blocks of 5 moving left to right, up to down. The collection graphic is sometimes mobile size (4 columns wide) and sometimes desktop sized (6 columns wide)
  7. A right arrow control should allow the user to scroll through the overflow content.

Community Page: Tablet

Acceptance criteria for community page on tablet

Same as desktop, except:

  1. Only 4 collections are shown at a time.
  2. The collection graphic is always 4 columns wide.
