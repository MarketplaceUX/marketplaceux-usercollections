Home Page → Community Page

The Community page is where users can view collections created by community members who have chosen to make their collections public.

Mobile

  1. Main Navigation Bar
    "Community" is a main navigation item listed after "Categories"
  2. Popular and New Tab
    By default, popular should be selected
  3. User Collections – What to Show?
    • Only show user collections that have 3 or more apps
    • If none of the apps in the collection are available in the user's region, the user's operator, or on their device (mobile use case) the user should not see that collection.
    • If an app in a collection is not available in the user's region, by the user's operator, or on their device (mobile use case), the user will not see that app though they will be able to see the rest of the apps in the collection
  4. Sharing and Upvoting
    • The user should be able to share a collection.
    • The user should be able to upvote a collection.

Desktop

  1. Main Navigation Bar
    "Community" is a main navigation item listed after "Categories"
  2. Page Title
    Only shown on Desktop and Tablet layouts
  3. Popular and New Dropdown Menu: Sort By
    • On Desktop and Tablet layouts, the Popular and New Tab is replaced by a Dropdown Menu.
    • By default, popular should be selected
  4. Create New Collection Link
    The create new collection call-to-action will take user to the create a collection screen.
  5. User Collections – What to Show?
    • Order: Collections are listed from most recently created to oldest, moving left to right, top to bottom.
    • Quantity: If the collection block is sized 4 columns wide, show 5 in one row. If block is sized 6 columns wide, show 4.
    • Size of element: Each collection block may be 4 or 6 columns wide
  6. Horizontal Scrolling Control
    A right arrow control should allow the user to scroll through the overflow content.

Tablet

  1. User Collections – What to Show?
    • Quantity: Only 4 collections are shown at a time.
    • Size of Element: Each collection block must always be 4 columns wide.

The rest of the features are exactly the same as the Desktop layout.
